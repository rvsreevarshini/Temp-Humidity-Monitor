# Temperature & Humidity Monitoring System (DHT22 + LCD)

This project measures temperature and humidity using the DHT22 sensor 
and displays values on a 16x2 LCD with I2C.

## Components
- Arduino Uno
- DHT22 Sensor
- 16x2 LCD with I2C
- Jumper Wires
